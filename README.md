# okashi-kingdom
お菓子王国

## Development
* [geth(Go Ethereum)](https://geth.ethereum.org/)
* [truffle](https://github.com/trufflesuite/truffle)

### Setup
```
yarn
```

### Compile
```
yarn truffle compile
```

### Migrate
```
yarn truffle migrate
```

### Console
```
yarn truffle console
```

### Test
```
yarn truffle test
```
